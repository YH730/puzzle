## Kurzbeschreibung
Erstellen eines Picture Puzzle Game (Schiebepuzzel)

Es ist ein Spiel, bei dem der Spieler aus 8 fragmentierten Teilen eines Fotos durch schieben das Bild in die richtige Ordnung bringt.
